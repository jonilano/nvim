# My Vim files

For Vim and Neovim

Plugins managed by [vim-plug].

<br>

## Install

```
make link     # symlink to ~/.vim
make restore  # install vim plugins
```

<br>

## Adding packages

```
vim ~/.vimrc     # add 'Plug 'x/y''
make install     # update ./bin/install
```

<br>

## Updating plugins

```
make update
```
